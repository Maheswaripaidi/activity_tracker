 Activity Tracker with NLP Summarization

📌 Description
This project is a desktop-based Activity Tracker that monitors user actions such as application usage, mouse clicks, and keyboard activity. It generates structured reports and uses NLP (TF-IDF) to create meaningful summaries.

🚀 Features
-  Tracks user activity (mouse, keyboard, applications)
-  Generates human-readable narration
-  NLP-based text summarization
-  Time tracking per application
-  Export report to file

 🛠️ Technologies Used
- Python
- PyQt6 (GUI)
- SQLite (Database)
- Scikit-learn (NLP - TF-IDF)
- Pynput (Input tracking)
- PyGetWindow (Window tracking)


▶️ How to Run
bash
pip install -r requirements.txt
python main.py
